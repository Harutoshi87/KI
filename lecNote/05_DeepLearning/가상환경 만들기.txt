※ 가상환경 만들기(ml-dl-nlp)


(가상환경 목록 확인) conda env list 또는 (conda info --envs)

(가상환경 생성) conda create -n ml-dl-nlp python=3.10

(생성된 가상환경으로 들어가기) conda activate ml-dl-nlp

(가상환경 나오기) conda deactivate

(가상환경 지우기) conda env remove --name 가상환경이름


(pip를 최신버전으로 업그레이드) python -m pip install --upgrade pip

(lib들 설치) pip install jupyter numpy==1.23.5 pandas==1.5.3 matplotlib==3.7.0 seaborn==0.12.2 scikit-learn==1.2.1

(tensorflow 설치) pip install tensorflow==2.10.0

(가상환경에 들어가서 jupyter 설치) conda install ipykernel

(Jupyter에서 선택가능한 커널로 등록) python -m ipykernel install --user --name ml-dl-nlp --display-name "ml-dl-nlp(ipykernel)"

(Jupyter 커널 목록 확인) jupyter kernelspec list

(Jupter 커널 목록에서 삭제) jupyter kernelspec remove display이름

(Jupyter lab 실행) Jupyter lab –notebook-dir=d:\ai\source\05_deepLearning
