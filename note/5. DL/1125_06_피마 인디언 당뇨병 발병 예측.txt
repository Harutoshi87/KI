# ✅ 1. **이진분류(Binary Classification)**

정답(클래스)이 **2개**인 문제

예:

* 스팸 / 정상메일
* 합격 / 불합격
* 양성 / 음성
* 사기 / 정상거래

출력 특징:

* 출력 뉴런 1개
* activation = **sigmoid**
* 예측값: 0~1 확률 → 0 또는 1로 판정

---

# ✅ 2. **다중분류(Multi-Class Classification)**

정답(클래스)이 **3개 이상**인 문제

예:

* 숫자(0~9) → 10클래스
* 과일 종류(사과, 바나나, 포도)
* 고객 등급(A/B/C/D)

출력 특징:

* 출력 뉴런 = 클래스 개수 만큼
* activation = **softmax**
* 예측값: 각 클래스 확률 → 가장 높은 index가 정답

---

# 🔥 핵심 비교표

| 구분     | 이진분류                | 다중분류                     |
| ------ | ------------------- | ------------------------ |
| 클래스 수  | 2개                  | 3개 이상                    |
| 출력층 뉴런 | 1개                  | 클래스 개수만큼                 |
| 활성함수   | sigmoid             | softmax                  |
| loss   | binary_crossentropy | categorical_crossentropy |
| 예측 방식  | 0/1                 | 확률이 가장 큰 클래스 선택          |
