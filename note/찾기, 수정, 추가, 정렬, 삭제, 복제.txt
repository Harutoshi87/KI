찾기 : 
	- 문자열 : find(), index() > extl. safe_index()
	- 리스트, 튜플 : index() > evtl. safe_index()
	- 딕셔너리 : 인덱싱['키'] 또는 get('')

추가 :
	- 리스트  	
			+ : 두 리스트 연결
			* : 리스트를 곱한 수만큼 반복
			append() : 요소 맨 뒤에 추가
			extend(리스트) : 리스트를 요소별로 맨 뒤에 추가
			insert(idx, insert할 값) : idx번째 인덱스 위치에 값을 추가

	- 딕셔너리
			리스트의 수정 방식 (인덱싱 또는 슬라이싱 사용) : 한 항목씩 추가
			update({}) : 여러 항목 추가

	-셋			
			add() : scala 데이터나 tuple 데이터를 하나 추가. list, dic, set 유형은 add()함수 사용 불가
			update() : list, tuple, dictionary, set 요소를 하나씩 따로 추가. dictionary 사용 시 key만 추가되고 value는 버림 (vgl. extend())

수정 :
	
	- 리스트
			인덱싱 또는 슬라이싱 사용
	- 딕셔너리
			리스트의 수정 방식 : 한 항목씩 수정
			update({}) : 여러 항목 수정



정렬 :
	- 리스트
			sorted() : 정렬 결과만 변경 (리스트 자체는 그대로)
			sort(); sort(reverse=True) : 리스트 자체의 정렬 변경(오름차순); 내림차순으로 변경
			reverse() : 리스트를 역순으로 변경


삭제 :
	- 리스트
			pop() : 가장 마지막 인덱스의 요소가 반환되고 삭제됨
			pop(idx) : idx번째 요소가 반환되고 삭제됨
			remove(data) : list 안의 특정 data 삭제됨 (반환은 안 함). 없는 데이터를 삭제 시 에러 발생! > evtl. safe_remove
			del 리스트 변수[idx] : idx번째 요소가 삭제됨 (반환은 안 함)
			clear() : 모든 데이터를 삭제
	-딕셔너리
			del dic['키'] 
			pop('키') : 키의 값을 반환하고 제거
	-셋
			remove() > evtl. safe_remove()
