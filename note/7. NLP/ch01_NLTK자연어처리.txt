# 명사만 추출 : NN, NNS, NNP, NNPS
# 태그	의미
# NN	단수 일반 명사 (common noun, singular)
# 예: dog, book
# NNS	복수 일반 명사 (common noun, plural)
# 예: dogs, books
# NNP	단수 고유 명사 (proper noun, singular)
# 예: John, London
# NNPS	복수 고유 명사 (proper noun, plural)
# 예: The Smiths, Americas
