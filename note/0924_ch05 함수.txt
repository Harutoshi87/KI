5장 함수

1. 함수의 정의 및 사용(def  함수명(매개변수...):)
	- Overloading(같은 이름의 함수를 매개변수를 달리 사용) 불가
	- Docstring(함수 맨 윗줄에 설명을 덧붙여서 사용을 쉽게 만듦) 사용 가능
	- 전역변수 vs 지역변수(lexical 특성)

2. 반환값이 필요하면 return 사용
	- return a, b (여러 개의 값을 튜플(a, b)로 만들어서 return)

3. 함수의 매개변수 
	- 매개변수가 기본값을 가질 때 : 기본값이 있는 매개변수는 함수 호출 시 선택적으로 사용 가능
	  * 매개변수의 기본값이 스칼라 변수일 때 : 기본값은 함수 정의 시점
	  * 매개변수의 기본값이 데이터구조 객체일 경우 : 기본값은 함수 호출 시점 
	- 키워드 인수 : 함수호출 시 인수가 매개변수 이름을 갖는 것 (매개변수 이름을 기입해야 하는 경우)
	- 튜플 매개변수 : 가변인수를 사용할 수 있음 *args(가변인수 구현 시 사용)
	- 딕셔너리 매개변수 **kwargs (여러 개의 키워드 인수를 딕셔너리로 형태로 사용)
	- 언패킹 : 튜플 인수 언패킹(*) vs 딕셔너리 인수 언패킹(**)
	
	-함수 정의 시 매개변수의 순서:
	  * 순서인수 > 튜플인수 > 키워드 인수 > 딕셔너리인수

4. lambda식
	- lambda 매개변수들 : 리턴값
	(funcname = lambda 매개변수들 : 리턴값 => funcname())
	- 용도 : 1) 함수의 매개변수 > 2) 함수를 return할 때
	  ex1. map(func, 가공할 vector)
	  ex2. filter(func, 추출할 vector)
	  ex3. sort(key=정렬 기준이 될 데이터를 return할 함수, reverse=T/F)
