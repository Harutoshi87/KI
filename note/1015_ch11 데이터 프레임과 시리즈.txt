3. 이름 (열, 행, 레벨) 지정하기 가능
	열 : columns =[]
	행 : index = []
	레벨 : columns = [[],[]]
	        columns.names = []
	        index = [[],[]]
	        index.names = []

4. 데이터 프레임의 부분 데이터 조회 방법

	df[열 이름] : 특정 열 조회
	df[조건] : 조건에 맞는 모든 행 조회
	df.loc[행 이름, 열 이름]
	행 이름, 열 이름 자리에 슬라이싱, 리스트 올 수 있음
	열 이름 생략 시 모든 열
	행 이름 자리에 조건 올 수 있음
	df.iloc[행 순번, 열 순번]
	행 순번, 열 순번 자리에 슬라이싱, 리스트 올 수 있음
	열 순번 생략 시 모든 열
